## 📌 About the Project

**HimaTechNo-Store** is a modern **tech e-commerce graduation project** designed to provide a professional online shopping experience for electronics and smart devices.
The project includes a complete storefront with multiple pages, product listings, wishlist, cart, checkout flow, and an admin dashboard interface.

---

## ✨ Features

* Responsive modern e-commerce design
* Home page with featured products and categories
* Shop page with product cards
* Product details page
* Wishlist page
* Shopping cart page
* Checkout page
* Contact page
* Login / Register pages
* User profile pages
* Admin dashboard pages
* Clean UI using Bootstrap and custom styling
* Interactive components using JavaScript

## ▶️ Getting Started

### Prerequisites

* Any modern web browser
* VS Code with Live Server extension (recommended)
* Git (optional)

### Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/allankhalaf/HimaTechNo-Store.git
   ```

2. Open the project folder in VS Code.

3. Run `pages/index.html` using **Live Server**.

---

## 📝 Notes

* The project is built as a front-end graduation project.
* Images are stored in the `images/` folder.
* The admin panel pages are stored in the `admin/` folder.
* The main user-facing pages are stored in the `pages/` folder.
