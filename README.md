## 📁 Project Structure
| Technology               | Usage                             |
| ------------------------ | --------------------------------- |
| **HTML5**                | Semantic markup                   |
| **CSS3**                 | Custom styling with CSS Variables |
| **Bootstrap 5**          | Grid system & components          |

Prerequisites
Any modern web browser
Git (optional, for cloning)

All scripts are internal (no external JS files)
Images stored in images/ folder
Admin panel in admin/ subfolder
